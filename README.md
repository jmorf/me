# me

This is my personal site.

## Building and running 

Development mode:

```bash
npm run dev
```

Build the app:

```bash
npm run build
```

Run locally:

```bash
npm run start
```


## Deploying

Just `commit`, and `push`, to auto-deploy with Github Actions.
